# Tori Proxy Requirement

记录 Tori Proxy 的 provider auth、credential、token refresh、通用 provider proxy 和 external connect 需求。

Tori Proxy 不记录 provider-specific 业务需求。Steam family/library、inventory、profile enrichment 等业务需求属于 Tori 的 provider module。
